Bootstrap is Mobile First.

Don't like it? Too bad.

BS2 used to have a "responsive" css file you could add in. BS3 is much more flexible and let's you really tailor the view to every device it will be viewed on.

What this means for you is:

  • You'll have to think about the mobile before/while thinking about the desktop experience.
  • You're stuff will look good on mobile without you having to think about it.
  • You're clients/customers will be happy because it looks good on their iWhatever.
  • You can start thinking about what content/media is most important, and add extra stuff in later.

Again, Bootstrap is not for everyone or every project, but before starting on your project think about how many mobile users you might lose.
