Bootstrap is a Framework.

So... what's a framework?

A framework is a library of abstracted (seperate) code meant to aid a developer in carrying out common tasks.

Examples:

I want to use...

PHP: CodeIgniter, CakePHP
JS: jQuery, Prototype, MooTools
Front-End HTML/CSS Development: Twitter Bootstrap, ZURB Foundation
A Grid System: Bootstrap, 960.js
Ruby: Ruby on Rails

Boostrap's JS is not a framework, it provides jQuery "plugins."

Bootstrap is not for every single project, but it can be used on so many things!
