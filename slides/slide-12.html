Add Bootstrap to your HTML5* document

* = you must declare your document as html5: <!DOCTYPE html>

The quick and dirty:

Add the minified CSS and JS in the head, and include jQuery before the Bootstrap JS

<script src="//ajax.googleapis.com/ajax/libs/jquery/1.10.2/jquery.min.js"></script>
<link rel="stylesheet" href="bootstrap-min.css">
<script src="bootstrap-min.js"></script>
Any styles you don't like from Bootstrap you'll have to overwrite.
Another way to do it would be to use the customizer script they have to download a version with only the bits you need. http://getbootstrap.com/customize/

The real way to do it:

This method takes longer, and involves using a LESS to CSS compiler tool, but it allows you to really customize Bootstrap.

  1. Download the Bootstrap source from Github.
  2. Find the LESS and JS files (feel free to use the JS min file if only wanting to change things in the CSS) and copy into your own project.
  3. Find a LESS compiler and import bootstrap.less, which refrences the other files.
  4. Use that generated (and hopefully minified) file as your bootstrap stylesheet.
Some good LESS compilers are: CodeKit (Mac only), Crunch App, and Simpless.
